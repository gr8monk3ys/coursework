# DeepLearning.AI NLP Specialization

## Overview

This directory contains assignments from the [Natural Language Processing Specialization](https://www.coursera.org/specializations/natural-language-processing) by DeepLearning.AI on Coursera. The specialization progresses from classical NLP techniques to state-of-the-art deep learning approaches.

## Courses

### Course 1: NLP with Classification and Vector Spaces

| Week | Assignment | Topic |
|------|-----------|-------|
| 1 | C1_W1_Assignment | Sentiment analysis with logistic regression on tweet data |
| 2 | C1_W2_Assignment | Naive Bayes classifier for tweet sentiment |
| 3 | C1_W3_Assignment | Word embeddings, cosine similarity, PCA visualization |
| 4 | C1_W4_Assignment | Naive machine translation using LSH (Locality Sensitive Hashing) |

### Course 2: NLP Processing with Probabilistic Models

| Week | Assignment | Topic |
|------|-----------|-------|
| 1 | C2_W1_Assignment | Building an autocorrect system with edit distance |
| 2 | C2_W2_Assignment | Parts-of-speech tagging with probabilistic models |
| 3 | C2_W3_Assignment | N-gram language models for auto-complete |
| 4 | C2_W4_Assignment | Training word embeddings from scratch |

### Course 3: NLP with Sequence Models

| Week | Assignment | Topic |
|------|-----------|-------|
| 1 | C3W1_Assignment | Deep N-grams with GRU/RNN using TensorFlow |
| 2 | C3W2_Assignment | Named Entity Recognition with LSTM networks |
| 3 | C3W3_Assignment | Question duplicate detection with Siamese networks |

### Course 4: NLP with Attention Models

| Week | Assignment | Topic |
|------|-----------|-------|
| 1 | C4W1_Assignment | Neural machine translation (English-to-Portuguese) with attention |
| 2 | C4W2_Assignment | Text summarization with transformer architecture |
| 3 | C4W3_Assignment | Question answering with T5 (Text-to-Text Transfer Transformer) |

## Technologies

- **Python 3** with Jupyter notebooks
- **TensorFlow / Keras** for deep learning models
- **NLTK** for text preprocessing and tokenization
- **NumPy / Pandas / Matplotlib** for data manipulation and visualization
- **Pre-trained word embeddings** (300-dimensional vectors)

## How to Run

1. Install dependencies: `pip install tensorflow nltk numpy pandas matplotlib`
2. Open any `.ipynb` file in Jupyter Notebook or JupyterLab
3. Run cells sequentially — each notebook is self-contained with inline instructions
