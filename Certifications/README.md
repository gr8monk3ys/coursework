# Certifications

## Overview

This directory contains coursework and assignments from online certification programs and specializations. These self-paced courses supplement formal university education with focused, industry-relevant skills in machine learning and natural language processing.

## Programs

### [DeepLearning.AI NLP Specialization](./DeeplearningAI%20NLP/)

A four-course specialization covering the full spectrum of NLP techniques, from classical methods to modern transformer architectures:

1. **NLP with Classification and Vector Spaces** - Sentiment analysis, word embeddings, and machine translation foundations
2. **NLP Processing with Probabilistic Models** - Autocorrect, POS tagging, N-gram language models
3. **NLP with Sequence Models** - RNNs, LSTMs, GRUs, and Siamese networks
4. **NLP with Attention Models** - Encoder-decoder architectures, transformers, and T5

**Tools:** Python, TensorFlow/Keras, NLTK, NumPy

### [Stanford Machine Learning](./Stanford%20Machine%20Learning/)

Andrew Ng's foundational machine learning course covering supervised learning, unsupervised learning, and best practices through 8 programming exercises in MATLAB/Octave.

**Tools:** MATLAB/Octave
