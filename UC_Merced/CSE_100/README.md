# CSE 100 - Algorithm Design and Analysis

This repository contains laboratory assignments for UC Merced's CSE 100 course, focusing on algorithm design and analysis. All labs are implemented in C++.

## Course Content

The course covers:
- Sorting algorithms (insertion, merge, heap, quick, radix sort)
- Divide-and-conquer strategies
- Dynamic programming
- Greedy algorithms
- Graph algorithms
- Hashing

## Lab Structure

| Lab | Topic | Description |
|-----|-------|-------------|
| **Lab_00** | Find Max/Min | Linear scan to find maximum and minimum values in an array |
| **Lab_01** | Insertion Sort | Sorts array by iterating and inserting each element into its correct position |
| **Lab_02** | Merge Sort | Recursive divide-and-conquer sort with merge step |
| **Lab_03** | Maximum Subarray | Divide-and-conquer solution for the maximum contiguous subarray problem |
| **Lab_04** | Heap Sort & Quick Sort | Two-part lab: heap sort using max-heap, and quick sort with partition-based pivoting |
| **Lab_05** | Radix Sort | Non-comparative sort using counting sort as a stable subroutine per digit |
| **Lab_06** | Hash Table (Chaining) | Hash table with separate chaining; supports insert, delete, search, and output |
| **Lab_07** | Rod Cutting (DP) | Bottom-up dynamic programming for optimal rod-cutting revenue |
| **Lab_08** | Matrix Chain Multiplication (DP) | Optimal parenthesization to minimize scalar multiplications |
| **Lab_09** | Huffman Coding (Greedy) | Builds optimal prefix-free codes using a priority queue |
| **Lab_10** | Strongly Connected Components | Kosaraju's algorithm with two DFS passes on original and transposed graphs |
| **Lab_11** | Minimum Spanning Tree | Prim's algorithm with priority queue for MST construction |
| **Lab_12** | Bellman-Ford Shortest Path | Single-source shortest paths with negative-weight cycle detection |
