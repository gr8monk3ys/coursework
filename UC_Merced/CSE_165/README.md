# CSE 165 - Object-Oriented Programming

This repository contains laboratory assignments for UC Merced's CSE 165 course focusing on object-oriented programming using C++.

## Course Content

The course covers:
- C++ Programming Fundamentals
- Object-Oriented Design Principles
- Memory Management
- Data Structures Implementation
- Advanced C++ Features

## Lab Structure

### Lab_01: Basic C++ Programming
- Area Calculations
- Vector Operations
- Prime Numbers
- Word Mapping

### Lab_02: Data Representation
- Structures
- Binary Operations
- String Manipulation
- Linked List Implementation

### Lab_03: Classes and Objects
- Point Class Implementation
- Friend Functions
- Stack Implementation
- Handle Classes
- String Vector Operations

### Lab_04: Pointers and References
- Proxy Classes
- Pointer Operations
- Array Management
- Text Processing

### Lab_05: Inheritance and Polymorphism
- Virtual Functions
- Instrument Classes
- Function Calls
- Point Finding Algorithms

### Lab_06: Memory Management
- Object Counting
- Dynamic Memory Allocation
- Pointer Stack (PStash)
- Heap Operations
- Placement New

### Lab_07: Static Variables and Pointer Functions
- Static pointers in functions and state preservation across calls
- Static variables with default parameters
- Static class members, static methods, inline member functions

### Lab_08: Templates, Operator Overloading, and Copy Semantics
- Copy constructor, copy assignment operator, destructor
- Friend functions and `ostream` operator overloading
- Class templates with non-type parameters, `[]` subscript overloading

### Lab_09: Inheritance and Exception Handling
- Multiple inheritance, method overriding, diamond inheritance resolution
- Custom exception classes, try-catch blocks, exception specifications

### Lab_10: String Manipulation, File I/O, and STL Containers
- String manipulation with iterators and escape sequences
- File I/O with `ifstream`, random number generation (Hangman game)
- STL `stack` container for Fibonacci sequence generation

### Lab_11: Linked Lists, STL Vectors, and Matrix Operations
- Singly linked list with pointer manipulation and list reversal
- STL `vector` with `push_back`, mathematical functions (`sin`)
- Matrix class with operator overloads (`+`, `*`, scalar `*`) using nested vectors

## Technologies Used
- C++
- Object-Oriented Programming
- Memory Management Tools
- Data Structures
- Algorithm Implementation

Each lab progressively builds upon previous concepts, creating a comprehensive understanding of object-oriented programming in C++.
